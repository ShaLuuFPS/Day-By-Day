# Day By Day

俯视角丧尸生存射击游戏 · Unity 6 + URP

## 玩法

双武器切换 + 体术近战 + 波次防守 + 肉鸽升级。

### 操作

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标 | 瞄准 |
| 左键 | 射击 / 近战（按住显示范围，松开攻击） |
| R | 换弹 |
| 1 / 2 | 切换武器槽 |
| Shift | Dash 冲刺 |
| E | 拾取地面武器 |
| ESC | 暂停 |

### 近战系统

- **按住左键**预览攻击范围（扇形蓝色面片），**松开**执行攻击
- 连击链：攻速冷却内连续攻击可打出连击段，超时重置
- 冷却期间不显示范围

### 换弹

- R 键随时换弹
- 弹夹打空 → 显示 `按R/左键换弹` → 0.5s 冷却后按/按住左键自动换弹

## 敌人

| 类型 | 特性 |
|------|------|
| 普通僵尸 | 靠近攻击，可配预警地面圈 |
| 爆炸僵尸 (Boomer) | 突进 + 自爆，预警期间不减速 |
| Boss | 高血量 |

## 升级系统（7 种）

击杀敌人获经验 → 升级 → 3 选 1 面板暂停游戏

| 升级 | 效果 |
|------|------|
| 穿透弹 | 子弹穿透 1 个敌人 |
| 分裂弹 | 命中后分裂 3 发（20% 伤害） |
| 尸体爆炸 | 击杀后 3m 范围 20 伤害（绿色爆炸） |
| 减速弹 | 命中减速 50%，持续 0.5s（影响冲刺） |
| 电磁弹 | 10% 触发连锁，30% 伤害，最多 5 连 |
| 击杀回耐 | 击杀恢复 20% 体力 |
| 吸血弹 | 伤害 3% 回血 |

## 技术方案

- **架构**：ScriptableObject 武器配置、IResettable 接口统一重置契约、静态事件解耦模块间通信
- **渲染**：URP 管线 + Canvas Scaler 锚点自适应布局 + TextMeshPro 富文本
- **输入**：Unity New Input System（键盘 + 鼠标）
- **特效**：预制体引用拖入，禁止 `Shader.Find` / `CreatePrimitive` 手搓材质
