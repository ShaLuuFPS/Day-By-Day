项目名称：《Day-By-Day》第一人称射击/生存 Demo（基于 Unity 6）

已实现核心玩法（直接对应你的 Commit）：

AI 系统：基于 Navigation 的敌人寻路、自动追踪玩家、范围攻击。

动态生成机制：敌人在场景中根据算法/时间点动态 Wave 刷新。

状态闭环：玩家血条 UI 渲染、受击判定、死亡结算及场景一键重置（Reset）。

思路：
俯视角3D的小游戏，大概是周围遍布着僵尸，僵尸数量与难度会随着天数（暂定2分钟一天）的增加而强大，角色能在地图上找到物资点，里面会刷新随机各类枪支、补给品。
