# Triage Labels

技能使用五个标准分类角色。此文件将这些角色映射到本仓库实际使用的 GitHub label 字符串。

| 技能中的标签 | 实际 label | 含义 |
|---|---|---|
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待提交者补充更多信息 |
| `ready-for-agent` | `ready-for-agent` | 描述完整，AI agent 可独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色（如 "打上 AFK-ready 标签"），使用此表中对应的 label 字符串。

编辑右列以匹配实际使用的标签词汇。
